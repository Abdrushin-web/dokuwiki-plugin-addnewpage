Dokuwiki Plugin: Add New Page
=============================

The add new page plugin for Dokuwiki: https://www.dokuwiki.org/plugin:addnewpage

# Syntax

    {{NEWPAGE[>namespace]}}

Where namespace is optional, and indicates the destination namespace for the new page.
